# Aula 01 — 11/05/2026

Introdução ao Backend com **Bun + Express**.

## Instalação

```bash
bun install
```

## Execução

```bash
bun run dev
```

Servidor disponível em `http://localhost:3000`.

## Rotas

| Método | Rota        | Resposta                                       |
|--------|-------------|------------------------------------------------|
| GET    | `/`         | `Olá Mundo`                                    |
| GET    | `/cadastro` | `vou realizar um insert no banco de dados`    |
